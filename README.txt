# Curso de python

Versión prototipo de un curso de python con uso intensivo de git

## Cómo hacer la tarea

* Has fork de este repo (solo una vez al principio del curso)
* Clona tu fork
* trata de correr los tests del día: `(env)$ python homework/2017-07-31.py`
* míralos fallar
* repáralos
* míralos funcionar
* crea un branch con tu código y publícalo
* has un PR para que pueda revisar la tarea
